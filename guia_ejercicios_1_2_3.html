01

Dimensionamiento sección circular

Dado un colector circular de diámetro conocido, pendiente y coeficiente de Manning, calcular el calado y y la velocidad media v para un caudal de diseño. Se resuelve por dos métodos: a) Gráficas de Thormann y Franke, b) Tabla de Thormann y Franke.

Datos del problema
Diámetro (Ø)
800 mm
Caudal de diseño Q
675 l/s
Pendiente j
6 ‰ (0.006)
Coef. Manning n
0.014
ℹ Se pide calcular el calado y (altura lámina de agua) y la velocidad media v mediante: a) Gráficas de Thormann y Franke  ·  b) Tablas de Thormann y Franke.
Paso previo — Caudal a sección llena (Q_ll)
1

Radio hidráulico a sección llena

Para tubería circular llena, el radio hidráulico RH es:

RH = Sm / Pm = (π·r²) / (2π·r) = r/2 = D/4
RH = 0.8/4 = 0.2 m
2

Fórmula de Manning-Strickler aplicada a sección llena

Qll = (1/n) · RH2/3 · j1/2 · S
Qll = (1/0.014) · 0.22/3 · 0.0061/2 · 0.5024
       donde S = π · r² = π · 0.4² = 0.5024 m²
Qll = 0.9511 m³/s = 951.1 l/s
Método a) — Gráficas de Thormann y Franke
Procedimiento gráfico
1

Calcular la relación Q/Qll

Q/Qll = 675 / 951.1 = 0.71

Esta relación es el valor que entra por el eje vertical de las curvas de Thormann y Franke (la curva de "caudal").

2

Entrar en la gráfica por la curva de Caudal

Con Q/Qll = 0.71 en el eje Y, trazar horizontal hasta cortar la curva de caudal. Bajar verticalmente al eje X para leer:

y/D → 0.62
3

Leer V/Vll desde el mismo punto

Desde y/D = 0.62, trazar vertical hasta cortar la curva de velocidad, luego horizontal al eje Y:

V/Vll → 1.08
4

Calcular calado y velocidad reales

y = (y/D) · D = 0.62 · 0.8 = 0.496 m ≈ 496 mm

Vll = Qll/Sll = 0.9511/0.5024 = 1.892 m/s
V = (V/Vll) · Vll = 1.08 · 1.892 = 2.043 m/s
Método b) — Tablas de Thormann y Franke
Procedimiento tabular
1

Buscar Q/Qll = 0.71 en la tabla

En la columna Q/Qll buscamos el valor 0.71. Se localiza directamente la fila:

y/DV/VllQ/Qll
0.6251.060.70
0.6331.060.71
0.6401.070.72
2

Obtener y/D y V/Vll

Para Q/Qll = 0.71 → y/D = 0.633 ; V/Vll = 1.06
3

Calcular calado y velocidad

y = 0.633 · 0.8 = 506 mm

V = 1.06 · Vll = 1.06 · 1.892 = 2.006 m/s
⚠ Las pequeñas diferencias entre ambos métodos (496 mm vs 506 mm) se deben a la precisión de lectura gráfica. El método tabular es más exacto. Ambos son válidos en la práctica profesional.

Colector de saneamiento — Año actual y horizonte

Dimensionar un colector de sección circular para un sector urbano con caudales actuales y de horizonte (25 años). Verificar las tres condiciones de velocidad.

Datos del problema
Q_med actual
0.05 m³/s
Q_min actual
0.02 m³/s
Q_p actual (neg.)
0.12 m³/s
Q_pl actual
1.00 m³/s
Q_med + 25 años
0.15 m³/s
Q_min + 25 años
0.06 m³/s
Q_p + 25 años
0.30 m³/s
Q_pl + 25 años
1.50 m³/s
Pendiente j
1 ‰ (0.001)
Coef. Manning n
0.0137
ℹ Red de alcantarillado → colector de drenaje + saneamiento. Se usa la Tabla de Thormann y Franke con j = 1‰ y n = 0.0137.
Paso 1 — Caudales de cálculo
1

Qmáx (para dimensionar el diámetro)

Se toma la suma de los máximos: aguas negras punta horizonte + pluviales horizonte.

Qmáx = Qp,n(+25a) + Qpl(+25a) = 0.30 + 1.50 = 1.80 m³/s
2

Qp (para comprobar V < 3 m/s en aguas negras)

Qp = Qp,n(+25a) = 0.30 m³/s
3

Qmin (para comprobar V > 0.6 m/s, arrastre)

Qmin = Qmin,n(actual) = 0.02 m³/s
← El peor caso es en el año actual, ya que hay menos caudal
Paso 2 — Cálculo del diámetro necesario
1

Aplicar Manning a sección llena para Qmáx

Despejando D de la fórmula de Manning para sección circular llena:

Q = (1/n) · (D/4)2/3 · j1/2 · (π·D²/4)

1.8 = (1/0.0137) · (D/4)2/3 · 0.0011/2 · π·D²/4

1.8 = 2.275 · D8/3

D = (1.8 / 2.275)3/8 = 0.916 m → Ø comercial = 1000 mm
Paso 3 — Comprobación de velocidades con Ø = 1000 mm
0

Qll y Vll para Ø = 1000 mm

Qll = (1/0.0137)·(0.25)2/3·(0.001)1/2·(π·1²/4) = 2.27 m³/s
Vll = Qll/S = 2.27 / (π·1²/4) = 2.89 m/s

Q/Qll=0.793 → y/D=0.69 → V/Vll=1.14

Qp,n → V < 3 m/s
(aguas negras)

V = 0.70 · 2.89
✓ 2.02 < 3 m/s   CUMPLE
3ª Condición
Qmin → V > 0.6 m/s
(arrastre de sólidos)
0.92 m/s

V = 0.33 · 2.89
✓ 0.92 > 0.6 m/s   CUMPLE
✅ Las tres condiciones se cumplen con Ø = 1000 mm. Este es el diámetro de diseño.

Dimensionamiento sección ovoide

Para una población de 302.400 hab con dotación de 250 l/hab·día, calcular caudales de aguas negras y dimensionar un colector ovoide unitario verificando las tres condiciones de velocidad.

Datos del problema
Población
302.400 hab
Dotación Dap
250 l/hab·día
Cretorno
0.8
Cp (punta)
1.4
Q pluviales
1.344 m³/s
Pendiente j
0.5 ‰ (0.0005)
Coef. Manning n
0.015
ℹ Ovoide normalizado: H = 1.5·D. Radios hidráulico y área: RH,ovoid = 0.193H − 0.193D  ·  Sovoid = 0.51H² − 0.51D²
Dimensiones comerciales posibles: 60×90, 90×135, 100×150, 120×180, 140×210 (D×H en cm)
Apartado a) — Caudales de aguas negras
1

Consumo medio diario

Consumo = 302.400 hab · 250 l/hab·día · (1 día / 86.400 s) = 875 l/s = 0.875 m³/s
2

Caudales de aguas negras (retorno = 80%)

Qm,n = CR · Consumo = 0.8 · 875 = 700 l/s = 0.70 m³/s

Qp = Cp · Qm,n = 1.4 · 700 = 980 l/s = 0.98 m³/s

Qmin = 0.2 · Qm,n = 0.2 · 700 = 140 l/s = 0.14 m³/s
Apartado b) — Dimensionamiento del colector ovoide
1

Caudales de cálculo (red unitaria)

Qmáx = Qp,n + Qpluviales = 0.98 + 1.344 = 2.324 ≈ 2.32 m³/s
Qp,n = 0.98 m³/s    Qmin = 0.14 m³/s
2

Predimensionamiento gráfico (suponer y/D = 0.8)

Entrando en la gráfica del ovoide con y/H = 0.8:

Q/Qll = 0.95 → Qll = Qmáx / 0.95 = 2.32/0.95 = 2.442 m³/s
V/Vll = 1.12
3

Calcular dimensión D del ovoide con Manning

Para ovoide con H = 1.5D → RH = 0.193·H = 0.193·1.5D = 0.2895D, S = 0.51·(1.5D)² − 0.51D² = 0.6375D²

Qll = (1/n) · RH2/3 · j1/2 · S

2.442 = (1/0.015) · (0.0005)1/2 · (0.193·1.5·D)2/3 · 0.51·1.5²·D²

D técnico = 1.517 m → Ovoide comercial 100×150 cm ó 120×180 cm
→ Se elige ovoide 120×180 cm (D=1.2 m, H=1.8 m) por ser el comercial inmediatamente superior al valor técnico.
4

Qll y Vll para ovoide 120×180 cm

RH = 0.193·1.8 − 0.193·1.2 = 0.1158 m (aprox 0.193·H con H=1.5D)
Usando RH = 0.193·D = 0.193·1.5 = 0.2895 m con D=1.2 → 0.347 m
S = 0.51·1.8² − 0.51·1.2² = 1.6524 − 0.7344 = 0.918 m²

Qll = (1/0.015)·(0.0005)1/2·(0.2895·1.2)2/3·0.51·1.5²·1.2² = ≈ 2.367 m³/s
Vll = Qll / S = 2.367 / (0.51·1.5²·1.44) = 2.367 / 1.65 = ≈ 2.062 m/s
Verificación de las 3 condiciones de velocidad

Se usa la Figura 3 (curvas de funcionamiento del ovoide) para obtener y/H y V/Vll a partir de Q/Qll.

1ª Condición
Qmáx → V < 5 m/s
2.30 m/s
Q/Qll = 2.32/2.367 = 0.98
→ Gráfica: y/H = 0.84 ; V/Vll = 1.12
V = 1.12 · 2.062
✓ 2.30 < 5 m/s   CUMPLE
2ª Condición
Qp,n → V < 3 m/s
1.897 m/s
Q/Qll = 0.98/2.367 = 0.41
→ Gráfica: y/H = 0.48 ; V/Vll = 0.92
V = 0.92 · 2.062
✓ 1.90 < 3 m/s   CUMPLE
3ª Condición
Qmin → V > 0.6 m/s
1.279 m/s
Q/Qll = 0.14/2.367 = 0.059
→ Gráfica: y/H = 0.23 ; V/Vll = 0.62
V = 0.62 · 2.062
✓ 1.28 > 0.6 m/s   CUMPLE
✅ Las tres condiciones se cumplen con el ovoide 120×180 cm. Solución válida.
